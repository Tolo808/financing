# admin

Next.js admin portal + shared backend API for the Tolo financing platform. See the [top-level README](../README.md) for architecture, setup, and deployment instructions.

Quick reference:

```bash
npm run dev              # start the dev server (http://localhost:3000)
npx vitest run            # run the finance-engine test suite
npx prisma migrate dev    # apply schema migrations
npx prisma db seed        # load demo data
```
